使用說明(忘記時點這裡)

一、第一次設定 Token(只做一次)

  1. 打開建立頁面:
    github.com/settings/personal-access-tokens/new
  2. Expiration 設長一點(例如一年)。
  3. Repository access → 選 Only select repositories → 只勾 HEX-qqdq/HEX。
  4. Permissions → Contents → 改成 Read and write。
  5. 按 Generate token,複製產生的 github_pat_...。
  6. 貼到本頁最上面的 Token 欄,勾「記住這台手機」。

二、開關頁面

  1. 每個頁面點「開放 / 關閉」。
  2. 「自動關閉日期」選了日期 → 到期當天晚上自動關;留空 = 不自動關,一直開到你手動關。
  3. 改完按底部金色「儲存全部設定」,約 1～2 分鐘生效。

三、要把 Token 給 Claude 時

  1. Claude 動你的 repo 前會先在對話問你同意。
  2. 你同意 → 按上面「📋 複製 Token」→ 回對話框貼上。
  3. Claude 只用這一次、做完即止。沒有你貼上,任何人都拿不到你的 token。

四、安全提醒

  1. 本控制台網址勿外傳(別人即使打開,沒 token 也改不了)。
  2. Token 過期後儲存會失敗,再開一把新的換上即可。
  3. 不想用了 → 到 GitHub 撤銷該 token;已上線的網頁照常運作,只是要再開關得換新 token。
